# 🌤 Weather App  

A simple and interactive weather app built using **HTML, CSS (Bootstrap), and JavaScript**, powered by the **OpenWeather API** to provide real-time weather updates for any city.  

## 🚀 Features  
- 🌎 Get real-time weather data for any location  
- 🔍 Search functionality to find weather details quickly  
- 🎨 Responsive design using Bootstrap  
- 🌡 Displays temperature, humidity, wind speed, and weather conditions  
- ☁️ Fetches accurate weather data using **OpenWeather API**  

## 🛠 Technologies Used  
- HTML, CSS (Bootstrap)  
- JavaScript  
- **OpenWeather API** (for real-time weather data)

## 📌 How to Use  
1. Enter the city name in the search bar.  
2. Click the search button to fetch the latest weather data.  
3. View the temperature, humidity, wind speed, and other weather conditions.  

## ⚙️ API Integration  
This project uses the **OpenWeather API** to fetch real-time weather data. To set up your own API key:  
1. Sign up at [OpenWeather](https://openweathermap.org/) and generate an API key.  
2. Replace `YOUR_API_KEY` in the JavaScript file:  
   ```javascript
   const apiKey = "YOUR_API_KEY";
   ```
